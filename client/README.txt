CLI Remote — connect this computer
===================================

Windows:  double-click  build.bat
Mac:      double-click  build.command   (if it won't open: right-click > Open)

A window opens and asks for consent. Type 'yes' and press Enter.
You are now connected — the operator can help until you close the window.
Closing the window ends control immediately.

Needs Python 3 installed (python.org). The launcher installs the rest.
